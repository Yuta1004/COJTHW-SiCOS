# COJTHW-SiCOS

**Si**mple **C**OJT **OS**

## 概要

自作CPU上で動かす自作OS  
[自作CPU COJTHW-CPUProj](https://github.com/Yuta1004/COJTHW-CPUProj)

## コマンド

### `make all`

`kernel.raw` `fs.raw` を生成

### `make run-shell`

ビルド用Dockerイメージのシェルを起動

### `make build-image`

ビルド用Dockerイメージの生成

### `make clean`

一時ファイルなど削除
